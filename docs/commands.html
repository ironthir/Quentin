Utility Commands


Weather

You can get current weather data from 200 000 cities around the world. Example: d!weather London

Weather forecast

Responds with weather forecast for given location. You need to specify which type of forecast you want. Hourly and daily forecast is available. Example: d!forecast 6h Oslo

Air pollution

Provides you with information about current air quality data for 200 000 cities around the world. Explanation of all values is available under d!pollution info. Example: d!pollution Warsaw

Current time

If you are curious about present time of the day in any city type d!time [city name] and Sverre will respond with some information about it.

Money exchange rates

You can check how many euros is $5 worth. Works with diffrent amounts and currencies. Example: d!moneyexchange 5 USD EUR

Yes/No poll

Create simple poll by typing d!poll [time in minutes] [question]

Search

Responds with top 10 results for any given query

Image

Similar to search, but responds with random image picked from the first 20 results. Example: d!search sahara

Movie

Search, but you get data only about movies. Example: d!movie Iron Man

Fun commands

Votekick

This command allows anyone to call a voting to kick given user from the server. Reason can be added, although it is not required.

Formula One results

Type d!f1results to see classification after last F1 race.

Play audio

Play command makes Sverre join a voice channel that you are connected to and play audio from YouTube URL for you. Usage: d!play [YT URL] There is also a disconnect command which kicks the bot out of your channel.

Commands for server administrators

Kick

Kicking a user has never been easier. Type d!kick [@user] [reason] to kick anyone out of the server. You need permission to kick members to do that. Reason is not required.

Ban

Banning is also very easy. Type d!ban [@user] [number of their messages to be removed] [reason] to ban anyone from the server. You need permission to ban members to do that. Reason and number of messages are not required.

Remove

You can remove up to 100 messages from any given channel. Only one hundred because there is a limit for Discord bots.
