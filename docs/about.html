Technologies used


Programming language

I chose JS by accident actually. I searched for tutorials on how to make Discord bot, and first I watched was based on it. I used it together with Node.js Do I regret it? No.

Discord Library

Biggest advantage of chosing JS was this library. It is very big and well described in docs, so I did not have any trouble working with it. This is the base for communicating with Discord API.

Hosting

Heroku was a good choice in my opinion. It allows me to host Sverre 24/7 for free. Easy to learn, transparent interface.

Manipulating images

I used Canvas to make profile pages. Easy to use, many features available.

API Marketplace

Rapid API provided me with many, easy to use APIs. I also used other APIs that are not listed here, but they are mentioned every time they are used by Sverre.

Database ORM

I used Sequelize ORM to create database for Sverre. Easy to use, but I needed to spend few hours to get along with it.
